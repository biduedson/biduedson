### 🚀 Projeto em destaque

**[CustomerApi — Clean Architecture, CQRS e Event Sourcing](https://github.com/biduedson/CustomerApi.ArquiteturaLimpa.CQRS.EventSourcing)**

API RESTful de gestão de clientes construída do zero com:
- Clean Architecture em 5 camadas
- CQRS com MediatR (Command e Query Handlers separados)
- Event Sourcing com EventStore próprio
- JWT + Refresh Token em Cookie HttpOnly
- SQL Server + MongoDB + Redis
- Docker Compose + GitHub Actions CI/CD
- 178+ commits
